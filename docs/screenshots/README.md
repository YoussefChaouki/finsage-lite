# Screenshots

Place PNG screenshots here. The README.md references these exact filenames:

| File | Description |
|------|-------------|
| `search-page.png` | Search page — empty state with search bar and controls |
| `search-results.png` | Search page — results with source cards and cited answer |
| `browse-page.png` | Browse page — filing selector and section navigation |
| `browse-content.png` | Browse page — section content view |
| `documents-page.png` | Documents page — filing grid with stats strip |

## How to add screenshots

1. Take a screenshot of the running app at `http://localhost:5173`
2. Export as PNG, name it exactly as listed above
3. Drop the file into this directory
4. Commit: `git add docs/screenshots/ && git commit -m "docs: add UI screenshots"`

Recommended resolution: **1440 × 900** or similar 16:9 widescreen.
The README renders images at natural width inside GitHub's Markdown viewer,
so screenshots taken at 1440 px wide look sharp on retina displays.
